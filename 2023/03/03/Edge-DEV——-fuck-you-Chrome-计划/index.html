Edge DEV浏览器

一、why edge instead of chrome?

​

​ 用的浏览器也挺多了，Chrome、Safari、edge、火狐，甚至最新的内测浏览器Arc都用过了。网络上的浏览器评测也看了不少，Mac上一开始主要使用Safari，主要是出于原生体验的考虑，后来发现部分网页兼容性不佳，就转成了chrome，没了兼容性问题，但是内存占用有点高（但是没卡过，于是也没有改用其他浏览器的动力）。

​ 后来为了windows和Mac的联动和New Bing的申请使用，开始使用edge，为了美观和稳定性，选择了Dev版本

二、edge版本选择

​ 只有三个测试版本才能使用新版bing和外观设计：

​ 选择了折中的DEV版本下载安装

​ 圆角设计和贴合Mac的外观非常让人舒适，更多的考虑到new bing的成功，可能会将bing带入Office 365 套餐，所以出于对未来使用的考虑，将edge作为主力浏览器了（战未来了属于是）

三、New Bing加入

​ 作为首批就加入waiting list的用户，等了快一个月都没收到通过邮件😭

​ 后来发现是申请时没有开启全局代理，地区不能设置为国内

​ 要注意的是，很有可能你已经通过bing的内测申请了，但是邮件发送有延迟，可以去bing/new中看一看，通过了是这样的

四、好用的快捷键

快捷键 功能
cmd + T 新标签页
cmd + Q 退出浏览器
cmd + W 关闭当前标签页
ctrl + tab 标签页切换
cmd + L 搜索框输入
